# tui-portfolio

A TUI (Terminal User Interface) portfolio app built with [Ink](https://github.com/vadimdemedes/ink) — React-like components for the terminal.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Or build then run:

```bash
npm run build
npm start
```

## Project structure

- `src/cli.jsx` — Entry point; renders the Ink app
- `src/App.jsx` — Root React component (edit this to build your portfolio)
- `src/components/` — Reusable Ink components (e.g. `Banner.jsx`)

Run in a **real terminal** (Ink needs a TTY for input). Exit: press `q` or `Esc`.
